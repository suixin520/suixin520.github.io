神奇的box-shadow

1. box-shadow简介

1.1 w3c上的简要说明

  box-shadow属性向2边框添加一个或者多个阴影。

box-shadow: h-shadow v-shadow blur spread color inset;

  • h-shadow： 必需，水平阴影的位置，允许为负值。
  • v-shadow：必需，垂直阴影的位置，允许为负值。
  • blur：可选，模糊距离
  • spread：可选，阴影的尺寸
  • color：可选，阴影的颜色
  • inset：可选，将外阴影改为内阴影。

2. 神奇的box-shadow

  不就是一个box-shadow的阴影属性嘛，为什么说它神奇呢？我们平时在使用box-shadow一般都是给边框加点阴影效果，使整个渲染效果看起来有立体感；另外我们可以利用box-shadow进行一些简单的绘图。

2.1 利用box-shadow做立体效果

其实立体效果很简单，给图片加上box-shadow属性后，设置相应的便宜距离和模糊值就好了

2.2 box-shadow绘制一朵云

如何绘制一朵云彩呢？我们可以利用多个box-shadow，使有的阴影重叠，多个不同大小的阴影组合起来形成一朵云彩。

2.3 利用box-shadow绘制一轮明月

利用绘制云彩的思路，我们也可以先绘制一个圆，然后将圆的背景色设置与大背景色相同，然后利用box-shadow绘制一个月亮形的阴影与背景色不一样即可。
